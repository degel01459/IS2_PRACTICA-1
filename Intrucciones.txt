
Configurar el Ambiente en WSL2

Configurar WSL2
Asegúrate de tener WSL2 configurado con una distribución de Linux (por ejemplo, Ubuntu).

Abre WSL2 desde la terminal:
wsl

Actualiza los paquetes:
sudo apt update && sudo apt upgrade -y

Clonar el Repositorio desde GitHub
Copia la URL de tu repositorio (en GitHub: Code > HTTPS).
Clona el repositorio en WSL2:
bash
Copiar
Editar
git clone <URL_DEL_REPOSITORIO>

Ingresa al directorio del repositorio:
bash
Copiar
Editar
cd <NOMBRE_DEL_REPOSITORIO>
